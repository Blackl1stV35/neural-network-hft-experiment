# Notebooks

This directory contains Jupyter notebooks for exploration and analysis.

## Suggested notebooks:

1. **01_data_exploration.ipynb** — Explore raw tick data, check distributions, visualize patterns
2. **02_feature_analysis.ipynb** — Analyze TA indicators, correlations, feature importance
3. **03_regime_analysis.ipynb** — Visualize regime labels, transition matrices, regime duration
4. **04_model_comparison.ipynb** — Compare CNN-LSTM vs Mamba vs TCN on validation set
5. **05_backtest_analysis.ipynb** — Deep dive into backtest results, trade analysis
6. **06_sentiment_exploration.ipynb** — Analyze FinBERT embeddings, news impact on price

## Quick start:
```bash
pip install jupyter
jupyter notebook notebooks/
```
